# Lerna AI Multiplatform SDK
The Lerna AI multiplatform SDK [[1]](#references) is built on Kotlin multiplatform and as such it can produce native SDK libraries for android, iOS, and Javascript. For more info please see the project's readme file. [[2]](#references)

It provides two options:
- either to use simple ML models like linear regression or
- to use advanced ML models by including the DNN library [[3]](#references)

## Execution

The Lerna SDK library can be downloaded via the maven repository. For usage please check the documentation [[4]](#references)


## Requires

- API that can be used by host app in order to interact with Lerna

## Required by

- FL API
- MPC Service via Gateway
- DNN Library (optional)
